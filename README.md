To successfully run my project you need to do the following steps. 

1. First you need to clone my project from the git repository
2. After cloning my project open the terminal go to the place you have downloaded the project and type in npm install 
3. After npm has installed then type in npm start 
4. Open a browser and type in localhost:4200 this is the port that the project should be running on
5. You can find test json files in the src/assets folder
